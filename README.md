### How to update the server binaries ?
execute:
```bash
npx altv-pkg <release | rc | dev>

# ON LINUX
chmod +x altv-crash-handler
chmod +x altv-server
```

### How to start the server ?
> Do not forget to update the server.toml token

execute:
```bash
# ON WINDOWS
altv-server.exe

# ON LINUX
./altv-server
```
